# jstart
chrome 插件，自定义起始页，自个专用
插件自定义了起始页，起始页就是一个搜索框，支持百度和 google 搜索，通知支持书签搜索。就这么简单没有别的多余的我不需要的东西。
支持在任何一个网页唤起起始页，通过快捷键可以快速唤起一个搜索框覆盖到当前网页上。
